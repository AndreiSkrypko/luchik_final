# Frontend - Детский центр «Лучик»

Next.js приложение для детского центра "Лучик".

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для production

```bash
npm run build
npm start
```

## Структура

- `src/app/` - страницы и layouts Next.js App Router
- `src/app/page.tsx` - главная страница
- `src/app/page.module.css` - стили главной страницы
- `public/` - статические файлы (изображения, иконки)
- `tailwind.config.ts` - конфигурация Tailwind CSS

## Дизайн

Все стили основаны на макете из Figma и полностью соответствуют цветовой палитре и структуре дизайна.
